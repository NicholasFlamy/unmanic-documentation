Installation on Unraid

Video Guide:#

Instructions:#

  1. Navigate to the Docker tab and scroll down to the bottom where it says "Template Repositories".
  2. Enter in a URL of https://github.com/Josh5/unraid-docker-templates in the "Template repositories" field.
  3. Click on the "Save" button

  1. Scroll back to the bottom of the Docker tab and then click on the "Add Container" button.
  2. Click on the "Template" drop-down menu and select the unmanic template.

  1. Modify the WebUI variables as desired:

Note: The Encoding Cache Directory is where files are written while they are being transcoded. Setting this to /dev/shm/unmanic will cause the files to be written to a RAM tmpfs. Set this to whatever you like.

  1. Scroll to the bottom and click the "APPLY" button

Once the image is downloaded you should see it appear in the Docker tab.

Nvidia GPU hardware encoding: [DEPRECATED]#

warning

Only use the below instructions if you are not running Unraid >= 6.9.0. The latest version of Unraid that ships with NVIDIA drivers already installed)

To use your NVIDIA GPU for hardware encoding in this container, follow these steps:

  1. Install the Unraid Nvidia Plugin to install a version of Unraid with the Nvidia Drivers installed. Linuxserverio Unraid NVIDIA Plugin

  2. Toggle this Docker Container template editor to "Advanced View".

  3. In the "Extra Parameters" field, add "--runtime=nvidia".

  1. Expand the template "Show more settings..." section near the bottom.
  2. In the "NVIDIA_VISIBLE_DEVICES" variable, copy your GPU UUID (can be found in the Unraid Nvidia Plugin. See that forum thread for details)

Once you have completed these steps, you should be able to use the h264_nvenc and hvec_nvenc encoders.
